
# Resume Screening System

## Description
A Python + SQL based resume screening system that parses resumes, matches job-required skills, and ranks candidates.

## Technologies
- Python
- MySQL
- SQL Queries

## Features
- Resume text extraction
- Skill matching
- Candidate ranking
- Database storage

## How It Works
1. Reads resumes from text files
2. Matches skills using keyword-based scoring
3. Stores scores in MySQL
4. Displays ranked candidates

## Use Case
Automates resume shortlisting for HR teams.
